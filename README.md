# Udemy-Analysis-Project

#Objective
-This Udemy platform analysis project analyze Udemy courses dataset to uncover trends in pricing, popularity, and subject distribution.

#Dataset
-Udemy courses dataset including course title, price, subscribers, reviews, and subject.

#Tools used
- Excel
- Power BI

#Key Questions
-Which courses have the largest number of participants (It turns out that web development, business finance, graphic design, and musical instruments have the most participants).
-We found that most participants in paid courses.
-We found that most courses, the longer they last, the higher the price.
